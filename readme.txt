Bulk edit and logging for stock related data of WooCommerce.

== Description ==

Stock Control allows you to bulk edit all stock related data of your products. It also logs all changes made to stock related data.

## Feature list

### Scope:

* Bulk edit stock related data.
* Log all stock changes.

== Installation ==

1. Upload `stock-control.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Products >> Stock Overview to start editing.
4. Navigate to Products >> Stock Log to view stock log entries.
